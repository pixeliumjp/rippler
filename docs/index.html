Rippler

このライブラリについて

WebGLを使用した美しい波紋エフェクトをWebページに追加するライブラリです。

クイックスタート

CDNを使う方法

HTML 
<div class="rippler" style="width: 100%; height: 400px; background-image: url('background.jpg');">
</div>
JavaScript 
<script type="module">
	import { Rippler } from 'https://cdn.jsdelivr.net/npm/rippler-js@latest/dist/rippler.min.js';
	const rippler = document.querySelector(".rippler");
	new Rippler(rippler);
</script>
